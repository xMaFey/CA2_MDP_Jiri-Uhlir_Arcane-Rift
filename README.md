OVERVIEW

Arcane Rift is a fast-paced 2D multiplayer arena game built using C++ and SFML with a client-server networking architecture.
Players connect over a network and battle in real time, using movement, positioning, and ranged attacks to defeat opponents.

_____________________________________________________________________________________________________

CONTROLS

 - Movement – Arrow Keys / WASD
 - Shoot – 1 / J
 - Dash – 2 / K
(can be changed in settings)

_____________________________________________________________________________________________________

OBJECTIVE

 - Defeat other players by reducing their health to zero using:
 - Movement and positioning
 - Ranged attacks
 - Dash ability for mobility

_____________________________________________________________________________________________________

MULTIPLAYER

 - Supports multiple players over a network
 - Uses a client-server model over TCP sockets
 - One player hosts the game, others join using the host’s IP
 - Real-time synchronization of player movement and actions
